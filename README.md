# Secret Santa Cloud

## Деплой
1. Зарегистрируйся в [Supabase](https://supabase.com)
2. Создай проект → включи Auth (Google)
3. Запусти `supabase.sql`
4. Залей на GitHub
5. В Render → New Web Service → подключи GitHub → добавь переменные:
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY`

Готово! Работает на `https://yourname.onrender.com`
